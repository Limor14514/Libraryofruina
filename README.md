## 命名规范
格式：`类型_分类_名称_版本_后缀`
示例：`Tex_Character_Liuhu_v01_png.png`

### 版本号规则
- `v01`：初版
- `v02`/`v03`：小修改迭代
- `v1.0`：正式定稿版
- **禁止**使用「最终版」「最新版」等模糊命名

---

## Git操作流程
1. 新增/修改素材 → 按规范命名 → 放入对应文件夹
2. 终端执行：`git add .`
3. 提交：`git commit -m "个人行为描述"`
4. 推送：`git push`
